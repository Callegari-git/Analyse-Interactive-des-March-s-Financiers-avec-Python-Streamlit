# Analyse-Interactive-des-March-s-Financiers-avec-Python-Streamlit
Application web interactive développée en Python (Streamlit) permettant d’analyser l’évolution des actions, cryptos et indices boursiers. L’utilisateur peut visualiser les prix, indicateurs techniques (RSI, moyennes mobiles, bandes de Bollinger), comparer plusieurs actifs et télécharger les données.
